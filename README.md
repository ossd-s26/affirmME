# **affirmME**
A browser extension that is a productivity-focused browser extension designed to motivate user through positive reinforcement. By tracking completed tasks using an external API, the extension generates encouraging affirmations whenever a user interacts with it, helping users stay motivated and consistent.

This project is being developed as an open-source collaborative effort, with a focus on clean UI design, seamless API integration, and user-friendly interactions.

## How It Works 
1. The user completes a task.
2. The task completed is tracked from an API.
3. When the user clicks on the browser extension, affirmMe generates a positive affirmation. 
4. The affirmation is displayed within the browser extension to motivate continous productivity. The user can click on a button to generate another positive affirmation if needed. 

## Installation Process 
- Fork the repository 
- Clone the repository git clone "url of the repository"
- cd affirmMe

## Load the Extension in Your Browser
- Open Google Chrome and go to: chrome://extensions
- Enable Developer mode (top right corner)
- Click Load unpacked 
- Select the project directory. The affirmMe extension should now appear in your browser
